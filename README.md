# Analizador de Ventas Comerciales - UTN

Trabajo Práctico Individual para la cátedra de Organización Empresarial 2026.

## 📊 Escenario Seleccionado
* **Escenario B:** Análisis de Ventas de una Pequeña Empresa.
* **Descripción del Dataset:** Registro analítico de ventas y movimientos comerciales (ID, fechas y montos diarios).

## 🗂️ Arquitectura del Repositorio
* `/datos`: Archivo fuente centralizado (`dataset.csv`).
* `/scripts`: Módulo algorítmico desarrollado en Python (`analisis_datos.py`).
* `/resultados`: Repositorio de métricas de rendimiento y gráficos descriptivos.

## 🚀 Guía de Reproducción
Para garantizar la reproducibilidad completa del análisis desde la raíz del proyecto local, corra:
```bash
python scripts/analisis_datos.py
```
